# Hướng Dẫn Sử Dụng Trước Khi Dùng
- Cài Đặt Nodejs( Nếu có rồi thì bỏ qua)
- Mở Project, khởi động terminal:
    ```
        npm i 
    ```
    và

    ```
        npm i mongoose
    ```
- Kiểm tra file config.json tại thư mục:
    ```
    src/config.json
    ```
    cấu hình localhost và connectionString theo máy của bạn
- Chạy project:
    ```
        node index.js
    ```
